# Product TDD

This layer owns durable technical truth that spans PartnerUp UI packages or
workspace-level workflows.

It does not own consumer usage snippets, generated output, or package-internal
implementation details.

## Files

- `unit-topology.md`: workspace units and ownership boundaries.
- `cross-package-contracts.md`: compatibility promises between web and UniApp.
- `generated-docs-and-skills.md`: generated references and their source of
  truth.
- `release-surface.md`: package release surface and public artifact policy.
